MultiStreamVlc Companion

Browser extension for the MultiStreamVlc desktop app

Configure Connection

Enter the host and port shown in the MultiStreamVlc desktop app's Settings window.


Endpoint: http://localhost:55432/

How to Use

  1. Open the MultiStreamVlc desktop app and start the companion listener via Settings
  2. Navigate to a page with a video stream and start playback
  3. Right-click anywhere and select Send to MultiStreamVlc, or press Ctrl+Shift+S
  4. The stream appears in the desktop app's dashboard — ready to play
  5. Click the toolbar icon to view history and resend or copy past streams
